# FAQ - Frequently Asked Questions

## General Questions

**Q: What technologies are used?**
A: Next.js, React, TypeScript, Tailwind CSS, and Framer Motion are used.

**Q: How do I run the project locally?**
A: Run `npm install` followed by `npm run dev` to start.

**Q: Where is the documentation?**
A: All documentation is in the `/docs` folder.

## Development Questions

**Q: How do I add a new page?**
A: Create a new file in the `app` directory with `page.tsx`.

**Q: How do I add new content?**
A: Edit the JSON files in the `content` directory.

**Q: How do I change the theme colors?**
A: Modify the color definitions in `tailwind.config.js`.

## Deployment Questions

**Q: Where is the site hosted?**
A: Deployed on Vercel.

**Q: How do I deploy updates?**
A: Push to main branch for automatic deployment.
