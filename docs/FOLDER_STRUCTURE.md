# Folder Structure

<!-- Last updated: 2026-03-13 -->

## Root Directory

- `/src` - Source code
- `/public` - Static assets
- `/content` - Content files
- `/docs` - Documentation

## Source Organization

- `/src/app` - Next.js app directory
- `/src/components` - React components
- `/src/lib` - Utility functions
- `/src/hooks` - Custom hooks
- `/src/types` - TypeScript types
