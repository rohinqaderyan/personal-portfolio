# Folder Structure

<!-- Last updated: 2026-01-31 -->

## Root Directory

- `/src` - Source code
- `/public` - Static assets
- `/content` - Content files
- `/docs` - Documentation

## Source Organization

- `/src/app` - Next.js app directory
- `/src/components` - React components
- `/src/lib` - Utility functions
- `/src/hooks` - Custom hooks
- `/src/types` - TypeScript types
