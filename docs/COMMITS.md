# Commit Message Guidelines

<!-- Last reviewed: 2026-07-13 -->

## Format

```text
<type>(<scope>): <subject>
```

## Types

- feat: New feature added
- fix: Bug fix
- docs: Documentation
- style: Formatting
- refactor: Code restructuring
- test: Testing
- chore: Maintenance

## Examples

- feat(auth): add login functionality
- fix(api): resolve timeout issue
- docs(readme): update installation steps
