# Next.js Patterns

## File-based Routing

- app/ directory structure
- Dynamic routes: [id]
- Route groups: (group)

## Data Fetching

- Server Components (default)
- Client Components ('use client')
- API routes in app/api

## Image Optimization

- Use next/image
- Set width/height
- Add alt text
