# Naming Conventions

<!-- Last reviewed: 2026-03-16 -->

## Files

- Components: PascalCase (Button.tsx)
- Utilities: camelCase (utils.ts)
- Hooks: camelCase with 'use' prefix (useTheme.ts)
- Types: PascalCase (User.ts)

## Variables & Functions

- Variables: camelCase
- Constants: UPPER_SNAKE_CASE
- Functions: camelCase
- Classes: PascalCase

## CSS Classes

- Use kebab-case
- Follow BEM when needed
