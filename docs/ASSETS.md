# Asset Management

<!-- Last reviewed: 2026-02-15 -->

## Images

- Store in public/ or assets/
- Use WebP format
- Provide multiple sizes
- Add lazy loading

## Fonts

- Self-host fonts
- Use font-display: swap
- Subset fonts
- Preload critical fonts

## Icons

- Use SVG
- Inline small icons
- Sprite sheets for many icons
