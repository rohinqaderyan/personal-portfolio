# Accessibility Checklist

<!-- Last updated: 2026-03-26 -->

## Keyboard Navigation

- [ ] All interactive elements focusable
- [ ] Logical tab order
- [ ] Visible focus indicators

## Screen Readers

- [ ] Semantic HTML
- [ ] ARIA labels where needed
- [ ] Alt text for images

## Visual

- [ ] Sufficient color contrast
- [ ] Text resizable
- [ ] No content flash
