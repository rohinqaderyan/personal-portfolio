# Content Writing Guide

## Markdown Syntax

```markdown
# Heading 1

## Heading 2

### Heading 3

**Bold text**
_Italic text_
[Link](url)
![Image](src)

- List item

1. Numbered item

`code`
```

## Frontmatter

```yaml
---
title: 'Post Title'
date: '2026-01-14'
description: 'Brief description'
tags: ['tag1', 'tag2']
---
```

## Best Practices

- Use clear headings
- Break into sections
- Add code examples
- Include images where helpful
- Keep paragraphs short
