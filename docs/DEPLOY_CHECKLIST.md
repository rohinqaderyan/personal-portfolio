# Deployment Checklist

## Pre-deployment

- [ ] All tests pass
- [ ] Build succeeds
- [ ] Environment variables set
- [ ] Database migrations run
- [ ] Dependencies updated

## Deployment

- [ ] Deploy to staging first
- [ ] Run smoke tests
- [ ] Check error monitoring
- [ ] Verify analytics

## Post-deployment

- [ ] Monitor logs
- [ ] Check performance metrics
- [ ] Test critical flows
