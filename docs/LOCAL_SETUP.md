# Local Development Setup

<!-- Last reviewed: 2026-04-27 -->

## Prerequisites

- Node.js 18+
- npm or yarn
- Git

## Installation

```bash
npm install
npm run dev
```

## Environment Setup

1. Copy .env.example to .env.local
2. Configure variables
3. Restart dev server
