# React Hooks Guide

<!-- Last reviewed: 2026-04-03 -->

## Built-in Hooks

- useState: Component state
- useEffect: Side effects
- useContext: Context API
- useMemo: Memoization
- useCallback: Function memoization
- useRef: DOM references

## Custom Hooks

- Extract reusable logic
- Follow naming convention (use\*)
- Can use other hooks
