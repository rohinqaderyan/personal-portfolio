# Profile Photo Instructions

<!-- Photo upload guide -->

Please add your professional photo with these specifications:

## File Details

- **Filename**: `profile-photo.jpg`
- **Location**: `/public/profile-photo.jpg`
- **Recommended Size**: 800x800px (square)
- **Format**: JPG or PNG
- **Max File Size**: 500KB

## Image Guidelines

- Professional headshot with good lighting
- Neutral or professional background
- High resolution and sharp focus
- Friendly, approachable expression

## Current Status

Your attached photo is ready to use - just save it as `profile-photo.jpg` in this directory!

## Usage

The photo will appear on:

- Homepage Hero section
- About page header
- OpenGraph social media previews
- Favicon/site icon (auto-generated)

## Additional Images Needed

1. **og-image.png** (1200x630px) - Social media preview image
2. **site-icon.png** (512x512px) - App icon/PWA
3. **favicon.ico** - Browser tab icon

You can use online tools like:

- [Favicon.io](https://favicon.io/) - Generate favicons
- [Canva](https://canva.com) - Create OG images
